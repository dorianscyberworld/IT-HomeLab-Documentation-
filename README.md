# IT Help Desk Homelab Documentation

## Overview
This project simulates a real-world IT help desk environment using Spiceworks Cloud Help Desk and GitHub documentation.

The goal is to practice ticket creation, ticket triage, troubleshooting, issue resolution, and professional IT documentation.

---

## Tools Used
- Spiceworks Cloud Help Desk
- GitHub
- Markdown Documentation
- IT Support Troubleshooting

---

## Skills Demonstrated
- Help Desk Ticket Management
- User Account Troubleshooting
- Password Reset Procedures
- Ticket Documentation
- Technical Troubleshooting
- End User Support
- Incident Resolution
