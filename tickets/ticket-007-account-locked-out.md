# Ticket 007 – Account Locked Out

**User:** Kevin Miles  
**Department:** Operations  
**Priority:** High  
**Category:** Account Access  
**Status:** Closed  

## Issue
User was unable to log into workstation after multiple failed login attempts. Account was locked.

## Troubleshooting Steps
- Verified user identity
- Confirmed account lockout
- Reviewed recent login attempts
- Verified no suspicious activity
- Unlocked user account
- Reset password
- Tested successful login

## Resolution
User account was unlocked and password reset successfully. User regained workstation access.
