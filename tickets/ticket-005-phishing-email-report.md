# Ticket 005 – Suspicious Phishing Email Report

**User:** Tanya Brooks  
**Department:** Sales  
**Priority:** High  
**Category:** Email  
**Status:** Closed  

## Issue
User reported receiving a suspicious email requesting login credentials through an unfamiliar link. User suspected the message was a phishing attempt and did not click the link.

## Troubleshooting Steps
- Verified report with user
- Confirmed user did not click the suspicious link
- Reviewed sender address and email content
- Identified phishing indicators such as urgency, unfamiliar sender, and credential request
- Advised user not to click links or download attachments
- Instructed user to delete the email and report future suspicious emails

## Resolution
Confirmed no account compromise occurred. User was educated on phishing indicators and the ticket was closed.
